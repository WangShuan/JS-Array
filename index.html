// 在變量中當你聲明 var b=a 就叫做值複製 但a跟b都是獨立存在的變量

數組元素的添加/刪除方式【shift、pop、push、unshift】

// shift：刪除第一個元素 並返回被刪除的那個元素
//pop：刪除最後一個元素 並返回被刪除的那個元素
// push：往最後面添加新的元素(push幾個就添加幾個) 並返回數組添加後的長度
// unshift：往最開頭添加新的元素(unshift幾個就添加幾個) 並返回數組添加後的長度

數組元素的排序：直接排序

以第一個數為基礎 然後拿第二個數與之做比較 如果比較大就放排好的數後面
第三個數就繼續跟前面排好的數做比較 比較小就往前排 比較大就放在最後面 依此類推

數組元素的排序：冒泡排序

從第一個數開始 拿第一個數往後面其他所有數比較
假設第一個數大於第二個數 就變成他放在二 然後他繼續跟三比較
如果三大 就換拿三跟四比較 如果還是三大 就繼續用三跟五比較
繼續往後推推推 看誰大誰就放到最後面 排完後開始第二次比較
一樣從目前第一個數拿出來跟後面的數比 直到沒得往後比為止
